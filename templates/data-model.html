Data Model

The database is designed to store details of each hero who is alive in the upcoming movie "Avengers: End Game".
The hero table consists of each hero who has one or more powers which is in turn represented in the power table.
A power is not strictly a description of a hero's abilities but can also be a trivia tidbit.
In the context of the design of this website, the collection of each hero's power is represented in a graphical comics-style format.
Each square then represents one of the powers of this hero in image form and in text form.

Hero table

Attribute Data type Description Validation
id Integer, Primary Key Unique identifier of each hero record Must be an integer, Not null
hero_name Variable Character (50 characters max) The name of the hero Not null
hero_image Variable Character (100 characters max) An image of the hero that serves as the background in the hero's detail page Not null
hero_description Text A description of the hero as viewed in the listing page of all heroes Not null
hero_color Variable Character (7 characters max) The main color motif associated with the hero; used for changing the border of the comics in the hero's detail page Not null

Power table

Attribute Data type Description Validation
id Integer, Primary Key Unique identifier of each power record Must be an integer, Not null
power_text Text A description of one of the powers of the hero; may also be a trivia tidbit about the hero Not null
power_image Variable Character (100 characters max) The filename of the image representing one of the powers of the hero Not null
hero_id Integer, Foreign Key The hero that is related to this power record Not null
